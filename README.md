# Student Records Search

A simple web application built using **HTML, CSS, and JavaScript** that displays student records in card format. Users can search for students by name using the search button, and matching records are displayed dynamically.

## Objective

The objective of this project is to practice JavaScript array methods by displaying student records dynamically and implementing a search feature using the **map()** and **filter()** methods.

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```text
Student-Records/
│── index.html
│── style.css
│── script.js
└── README.md
```

## Features

- Display student records in responsive cards.
- Store student data using an array of objects.
- Generate student cards dynamically using the **map()** method.
- Search students by name using the **filter()** method.
- Search is performed when the **Search** button is clicked.
- Clean and responsive user interface.

## How to Run

1. Clone or download this repository.
2. https://github.com/kjwashim/Task-16-Map-Filter
3. Open the project folder.
4. Open `index.html` in your web browser.
5. Enter a student's name in the search box.
6. Click the **Search** button to filter the student records.
